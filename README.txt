1. Открыть файл "install.bat" в режиме редактирования.​
2. Изменить команду pip install -r requirements.txt так, чтобы она работа на вашей системе:​
	- На внешних ПК работает изначально.​
	- На АРМ/ВАРМ нужно прописать pip install через токен SberOSC​
3. Сохранить изменения, запустить install.bat. Произойдет установка виртуальной среды и запуск сервера.​
4. После однократной установки, можно запускать сервер файлом start.bat, пропуская шаги.​
5. Когда сервер запущен, сайт открывается по адресу http://127.0.0.1:8000​
